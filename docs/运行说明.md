# 运行说明

## 1. 检查项目资源

```powershell
cd F:\project\智能媒体工作坊
pip install -r requirements.txt
python src/main.py --task check
```

应看到三类 Data 数量均为 2880，并能列出 ResNet50、Swin_T 本地资源。

## 2. 裁切数据

已有裁切数据时只重建索引：

```powershell
python src/crop_pages.py
```

原始扫描页已从精简提交版中删除。需要重新裁切时，应先恢复三类原始扫描页，再运行：

```powershell
python src/crop_pages.py --force
```

裁切参数在 `config/crop_config.json`，包括起始坐标、行距、单元格尺寸、左右栏坐标和跳过参考列后的抄写列。

## 3. 运行选题 4

```powershell
python src/main.py --task topic4 --skip-feature
```

输出在：

```text
outputs/topic4/figures
outputs/topic4/tables
outputs/topic4/metrics
```

## 4. 运行选题 5

```powershell
python src/main.py --task topic5
```

输出在：

```text
outputs/topic5/figures
outputs/topic5/tables
outputs/topic5/metrics
```

## 5. 运行全部

```powershell
python src/main.py --all --skip-feature
```

`--skip-feature` 表示直接复用已有 ResNet50 PCA、Swin_T PCA、形态学特征和融合特征缓存；若删除这些缓存，再运行时会提示缺少对应文件。

## 6. 关键说明

- 原始 `基础案例A-视觉分析2/视觉分析.py` 已保留，备份为 `视觉分析_backup.py`。
- 正式整理脚本未强制联网下载模型。
- 精简提交版仅保留正式实验使用的 ResNet50 与 Swin_T 权重；ViT_B16、CLIP、DINOv2 未保留。
- 当前统一输出使用 `GroupShuffleSplit` 按 `page_id` 分组划分训练/测试，降低同一扫描页样本同时进入训练集和测试集的风险。
