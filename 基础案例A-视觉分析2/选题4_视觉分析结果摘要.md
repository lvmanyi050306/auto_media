## 视觉分析流程

原版 `视觉分析.py` 的思路是：读取 `Data` 分组图像，提取深度视觉特征，做 PCA 主成分分析，再做 t-SNE 二维降维可视化。

当前机器没有 GPU，原版 ViT/CLIP 逐张推理过慢，因此本次选题4使用本地预训练 `ResNet50.pth` 跑同一套视觉分析流程。实际运行脚本为：

```text
视觉分析_选题4_ResNet50.py
```

运行命令：

```powershell
$env:PYTHONPATH=(Resolve-Path '.pydeps').Path
python '视觉分析_选题4_ResNet50.py'
```

## 核心结果

1. PCA 前 10 个主成分中，9 个主成分在三组之间达到显著差异，说明端正易读、美观、极致扭曲三种书写目标在深度视觉特征空间中确实存在系统性差异。

2. 随机森林使用 ResNet50 的 PCA 特征进行三分类，准确率为 0.7407，Macro F1 为 0.7372。说明模型可以较稳定地区分三种书写目标。

3. 三组两两距离如下：

| 对比 | 前10 PC中心距离 | t-SNE中心距离 | PC1 Cliff's delta |
|---|---:|---:|---:|
| 端正易读 vs 美观 | 12.964 | 24.728 | 0.298 |
| 端正易读 vs 极致扭曲 | 14.127 | 31.560 | 0.335 |
| 美观 vs 极致扭曲 | 3.555 | 6.889 | 0.050 |

4. 差异大小排序为：

```text
端正易读 vs 极致扭曲 最大
端正易读 vs 美观 次之
美观 vs 极致扭曲 最小
```

## 可写入报告的结论

基于 ResNet50 深度视觉特征、PCA 主成分、t-SNE 可视化与分类实验，三种书写目标下的字迹变形存在显著差异。端正易读与极致扭曲之间差异最大，说明极致扭曲会显著改变符号的整体视觉结构；端正易读与美观也存在明显差异，但美观与极致扭曲之间有较多重叠，说明“美观”和“扭曲”并非完全相反的两个视觉类别，二者都可能包含笔画夸张、结构调整和局部变形，只是扭曲程度与结构破坏更强。
